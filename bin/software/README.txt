  How are these tools used for packaging?
  ---------------------------------------

  1) The "Batch 2 Execute Converter" is used for converting the batchfiles of the folder
      "\clansuite\serverpacks\clansuite4w_builder\bin\batch" to executables.
      
  2) "IcoFX" is used to apply the clansuite-icons to the executables.
  
  3) The folder "/webserver" is packed by Ispack to a self-executable Windows-Installer-Package.
